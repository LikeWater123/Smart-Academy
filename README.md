# 智慧学堂微信小程序

## 创作故事

我曾经给高中的学生带过课，是一门人工智能课外课，我在跟学生教学的过程中能明显感觉到他们对于现在新技术的热情，同时也察觉到我们除了在课堂，课外几乎没有交流，并且在上课的时候也很难顾及到所有学生。我跟其他老师也聊过，他们说："一个班50个孩子，我真的顾不过来。"这位老师每天花大量时间批改作业，却很难从中了解每个学生真正哪里不会、为什么不会。更让她无奈的是，很多学生成绩下滑，并不是因为不努力，而是遇到了家庭变故、心理压力等问题——但这些，分数上看不出来。

当前的教学现实中，师生之间缺乏有效的沟通桥梁。教师被繁重的批改任务淹没，学生的问题被冰冷的分数掩盖，家长对孩子的真实学情一无所知。我希望用AI技术改变这一切——让AI承担数据分析的苦活，让教师回归"育人"的本质。

智慧学堂由此诞生。它不仅是一个作业管理工具，更是一个连接教师与学生的智慧纽带：AI自动分析学生的薄弱知识点，生成个性化学习方案，同时通过暖心提醒关注学生身心健康，帮助教师真正做到因材施教、言传身教。

## 功能亮点

1. **AI自动诊断学生薄弱知识点**
2. **一键生成个性化学习方案**
3. **智能暖心提醒关注身心健康**
4. **班级学情数据看板辅助决策**
5. **连接师生从教书到真正育人**

## 使用场景

### 场景 1：日常作业与学情分析
教师通过智慧学堂发布作业，AI自动批改并分析每个学生的答题情况，诊断出薄弱知识点，形成班级学情报告。

### 场景 2：个性化学习方案推送
基于学生的薄弱知识点，AI智能生成个性化的学习方案和练习题目，帮助学生有针对性地提升。

### 场景 3：师生暖心沟通
通过暖心提醒功能，教师可以关注学生的身心健康状态，及时给予关怀和指导，建立良好的师生关系。

### 场景 4：家校协同育人
家长可以通过小程序了解孩子的学习情况和身心健康状态，与教师共同关注孩子的成长。

## 项目结构
- `smart-classroom/` - 智慧学堂微信小程序
  - `cloudfunctions/` - 云函数目录
    - `ai-service/` - AI 服务云函数
    - `assignment/` - 作业管理云函数
    - `login/` - 登录云函数
    - `quiz/` - 测验云函数
  - `miniprogram_npm/` - 小程序依赖包
  - `pages/` - 页面目录
    - `login/` - 登录页面
    - `teacher/home/` - 教师首页
    - `teacher/create/` - 创建作业页面
    - `teacher/dashboard/` - 数据看板页面
    - `student/home/` - 学生首页
    - `student/quiz/` - 测验页面
    - `student/plan/` - 学习方案页面
    - `student/reminder/` - 提醒页面
  - `app.js` - 小程序入口文件
  - `app.json` - 小程序配置文件
  - `app.wxss` - 全局样式文件
  - `project.config.json` - 项目配置文件

## 开发说明
1. 使用微信开发者工具打开 smart-classroom 目录
2. 在云开发控制台中创建对应的云函数
3. 上传云函数到云端
4. 在本地进行调试
